HTML Tags

HTML is a markup language and it makes use of various tags to format content. Tags are enclosed within angle braces like this:<Tag-name>

Examples: <p>,<img>,<table><ul> etc.....

Except few tags, most of the tags have their corresponding closing tags. For example, <html> has its closing tag </html> and <body> tag has its closing tag </body> etc.
